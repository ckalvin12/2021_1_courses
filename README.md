109學年度第二學期期中、期末成績輸入及繳交注意事項
請轉知所系(學程)所屬教師

受文者：本校日間部授課教師
主旨： 109學年度第二學期期中、期末成績輸入及繳交注意事項
說明：
一、所有科目(含未考試學科，例如實驗、實習等)均須要輸入期中成績。

二、成績輸入網址http://120.114.51.69/itwt/login.asp，或進入行政單位→註冊組→教師成績系統。

三、學校網路成績輸入開放日期及時間如下：
1.期中考成績輸入：110年4月19日(一)上午8時起至110年4月30日(五)下午5時止。期中成績比例請勿調整為100%，否則會造成課程成績顯示錯誤。
2.畢業班畢業考(含研究所畢業班)成績輸入：110年5月31日(一)上午8時起至110年6月7日(一)下午5時止。為配合6月19日(六)畢業典禮，請畢業班授課老師務必於規定時間內完成成績輸入，以免影響畢業生權益。
3.期末考成績輸入：110年6月21日(一)上午8時起至110年7月2日(五)下午5時止。研究所(非畢業班)期末考成績輸入：110年7月30日(五)前完成。

四、請任課老師輸入成績後務必點選傳送-校對-產生確認單(期中成績確認單請自行列印留存，不需繳交至註冊組)。
註：請老師至授課選單檢視該科目功能狀態是否從選擇變成「已完成」。若狀態顯示「已完成」則表示該科目成績輸入完成；若仍顯示「選擇」，則表示該科目成績尚未輸入完成。

五、各項成績所佔百分比，可由任課教師自行訂定，並填入成績記載表內。

六、學生總成績不及格，請以紅筆填寫，以示區別，若有畫線刪除或塗改，請任課老師務必簽名或蓋章。

七、期末總成績輸入完畢後，請印出「成績輸入作業確認單」，並詳細核對及簽名後，連同「教師成績記載表」送教務處註冊組。

八、注意事項：
1.老師欲更正學生成績，須經系、院及教務會議同意後，方能更改；請任課老師成績輸入後，需再詳細核對「成績輸入作業確認單」與手寫的「教師成績記載表」，務必一致。
2.對網路成績輸入有任何疑問，請與教務處註冊組聯絡。
